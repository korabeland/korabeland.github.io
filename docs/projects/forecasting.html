Enrollment Forecasting System

Using Claude Code to build a machine learning forecasting system that predicts student enrollments, enabling better resource planning and budgeting for educational institutions.

The Challenge

Educational institutions need accurate enrollment forecasts to plan resources, hire faculty, and set budgets months in advance. With a high volume of annual leads and multiple intake periods, partners, programs, and marketing channels, manual forecasting was time-consuming, error-prone, and couldn't provide the granularity needed for effective planning.

I used Claude Code to design a systematic approach that could replace hours of manual spreadsheet work with automated, data-driven predictions.

The Goal

Build a forecasting system that predicts enrollments for the next 3 intake periods across multiple dimensions: partner institutions, academic programs, and marketing channels.

Target accuracy: Within 5% margin of error, with confidence intervals for conservative/optimistic scenarios.

Development Approach

Rather than building a full production system immediately, I used Claude Code to design a phased approach that validates feasibility before major investment.

How It Works

Working with Claude Code, I designed a forecasting approach that combines historical patterns with current pipeline data.

Core Methodology

  1. Analyze historical patterns: Look at how past leads converted over time
  2. Score current pipeline: Estimate conversion likelihood for leads already in the system
  3. Project future volume: Forecast how many new leads we'll receive
  4. Generate predictions: Combine all factors to predict enrollments with confidence ranges

Key Features

Multi-Dimensional

Forecasts broken down by partner institution, academic program, and marketing channel

Confidence Ranges

Conservative, base, and optimistic scenarios for risk-aware planning

Data Quality Checks

Automated validation with clear reporting of any issues

Dashboard Ready

Outputs formatted for Power BI visualization and stakeholder reporting
