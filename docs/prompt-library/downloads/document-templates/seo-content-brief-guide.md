# SEO Content Brief Guide

This folder contains templates and evaluation frameworks for technical documentation and SEO-optimized content creation.

## Files in This Folder

- **technical_documentation_how_to.md** - Production-ready template for creating clear, well-structured technical documentation that serves as both reference material and learning tool.
- **seo_content_brief_quality_evaluator.md** - Quality evaluation framework for assessing technical content and SEO briefs on dimensions including keyword optimization, user intent alignment, and structure effectiveness.
- **seo-content-brief-guide.md** - This file; guidance on creating technical documentation that works for both readers and search engines.

## How to Use

1. Use **technical_documentation_how_to.md** to generate your technical guide or documentation
2. Evaluate your content with **seo_content_brief_quality_evaluator.md** to ensure it meets quality, clarity, and SEO standards
3. Remember: Great technical content serves **human readers first**, search engines second

## Key Principle

Technical documentation must balance **accessibility for beginners** with **depth for experts**. Structure, clear headings, and progressive complexity are your tools.
