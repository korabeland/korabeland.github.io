# Technical Content Folder Guide

This folder contains templates for creating high-quality technical documentation and SEO-optimized content.

## Files in This Folder

- **technical_documentation_how_to.md** - Production-ready template for creating clear, structured technical documentation that serves as both reference material and learning resource.
- **seo_content_brief_quality_evaluator.md** - Quality evaluation framework for assessing SEO content briefs on dimensions including target keyword optimization, user intent alignment, and content structure.

## How to Use

1. Use **technical_documentation_how_to.md** to generate clear, well-structured technical guides
2. Evaluate your technical content with **seo_content_brief_quality_evaluator.md** to ensure it meets quality and SEO standards

## Key Principle

Technical documentation must balance **clarity for beginners** with **depth for experts**. Structure is your primary tool.
